# YARN SHOP APP

Det här är ett litet Pythonprogram som låter en besöka en garnaffär i egenskap av kund.
I affären kan man välja att
• Kolla igenom de garner som finns i butiken (utifrån färg, tjocklek, fibertyp)
• Köpa ett specifikt sorts garn
• Kontrollera vad man har i sin kundkorg
• Betala för sina varor
• Spara information om sin shoppingtur (kvitton och korghistorik)
Och sånt.

Programmet körs med fördel i terminalen i ett Pythonkompatibelt program, t.ex. VSCode.
1. Ladda in repot / filerna
2. Välj mappen 'app' och hitta filen 'shoppe.py'
3. Kör filen (i terminal: python app/shoppe.py)
4. Följ de (stundvis) inlevelsefulla instruktionerna och navigera affären därefter
5. Återbesök affären ('PLAY AGAIN') eller avsluta programmet
Tester:
Testerna ligger i mappen 'tests' och körs i terminalen med kommandot 'pytest' (utan ').
